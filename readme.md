# creer un microservice avec node.js et micro

## prerequis

- avoir node.js install√ sur sa machine: [site officiel de node.js](https://nodejs.org/fr/)

## initialiser le projet node.js

```
npm init
```
suivre le wizard pour creer les fichiers necessaires au demarrage d'un projet


## installation de micro

```
npm install -save micro
```

## ajout de micro dans package.json
ajouter le script start:
```json
"start": "micro"
```

## creer un fichier index.js
a la racine du projet, 
```
touch index.js
```
... and here we go!
